# スタエフ配信 紹介LP

stand.fm の配信を紹介するランディングページです。
静的HTML/CSS/JavaScript のみで構成しており、ビルド不要・依存パッケージなしで
Vercel にそのままデプロイできます。

## 構成

ヒーロー → 共感 → 内容 → CTA（stand.fm を聴く）の4セクション。

| ファイル | 役割 |
| --- | --- |
| `index.html` | ページ本体。文言の編集はここだけで完結します |
| `styles.css` | デザイン。冒頭にカラーパレットをまとめてあります |
| `script.js` | スクロール時のフェードイン演出のみ |
| `vercel.json` | Vercel の設定（クリーンURL・セキュリティヘッダー） |

## 公開前に差し替えが必要な箇所

`index.html` を **「【」で検索** すると、すべて見つかります。

- `【チャンネル名】` — ヘッダー / タイトル / フッターの3か所
- `【stand.fmのURL】` — CTAリンク3か所。**未設定のままだとボタンが機能しません**
- `【週2回・朝】` — 配信頻度
- `【10】` — 1話あたりの長さ（分）
- `【配信者名】` — フッターの著作権表記

あわせて、OGP画像 `ogp.png`（1200×630px）をこのフォルダに置き、
`index.html` の `og:image` を絶対URL（`https://〜/ogp.png`）に書き換えてください。

## カラーパレット

`styles.css` 冒頭の `:root` で定義しています。

| 用途 | 名称 | 値 |
| --- | --- | --- |
| メイン | ペールブルー | `#B8D4E3` |
| サブ | ミスティホワイト | `#F5F7F9` |
| アクセント | ソフトグレー | `#C7CCD1` |
| アクセント | ダスティブルー | `#8FA8B8` |
| アクセント | パールホワイト | `#F4F1EA` |
| CTA・リンク | コーンフラワーブルー | `#7B9ACC` |

### ボタンの色について

コーンフラワーブルー `#7B9ACC` に白文字を載せるとコントラスト比が 2.9:1 となり、
WCAG の基準（4.5:1）を大きく下回って読みにくくなります。
そのためボタンの塗りには、同系統で暗さを足した `--cta-deep` `#4A6DA7`（5.2:1）を
使用しています。罫線や見出しの装飾には `#7B9ACC` をそのまま使っています。

より淡い印象にしたい場合は `styles.css` の `--cta-deep` を `#7B9ACC` に変更できますが、
可読性は下がります。

## ローカルでの確認

`index.html` をブラウザで直接開けば確認できます。
CSSを編集しても見た目が変わらない場合は、ブラウザのキャッシュが原因です
（スーパーリロード: `Cmd + Shift + R`）。

## デプロイ

GitHub の `main` ブランチに push すると Vercel が自動でデプロイします。
