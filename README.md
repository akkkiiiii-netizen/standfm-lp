# ランディングページ

静的HTML/CSS/JavaScript で構成したLPです。ビルド不要・依存パッケージなしで、
そのまま Vercel にデプロイできます。

## ファイル構成

| ファイル | 役割 |
| --- | --- |
| `index.html` | ページ本体。全セクションのテキストはここを編集します |
| `styles.css` | デザイン。冒頭の「デザイントークン」で色を一括変更できます |
| `script.js` | スクロール表示演出のみ（任意） |
| `vercel.json` | Vercel の設定（クリーンURL・セキュリティヘッダー） |

## ローカルでの確認

`index.html` をブラウザで直接開けば確認できます。

## 編集の入口

1. **文言**: `index.html` 内の「ここに〜」というプレースホルダーを差し替える
2. **色**: `styles.css` 冒頭の `--brand` を自社カラーに変更する
3. **OGP画像**: `ogp.png`（1200×630px）をこのフォルダに置く
4. **お問い合わせフォーム**: `index.html` の `<form action="">` に送信先を設定する
   （静的サイトのためサーバー処理はありません。Googleフォーム / Formspree /
   Tally などの外部サービス、または Vercel Functions の追加が必要です）
